## Installation

<Steps>
  <Step title="Install Doom Emacs">
    ```bash
    git clone --depth 1 https://github.com/doomemacs/doomemacs ~/.emacs.d
    ~/.emacs.d/bin/doom install
    ```
  </Step>
  
  <Step title="Copy Configuration Files">
    Place the configuration files in `~/.doom.d/`:
    - `config.el` - Main configuration
    - `init.el` - Module selection
    - `packages.el` - Package declarations
  </Step>
  
  <Step title="Sync Configuration">
    ```bash
    ~/.emacs.d/bin/doom sync
    ```
  </Step>
  
  <Step title="Restart Emacs">
    Launch Emacs to apply the new configuration
  </Step>
</Steps>

## Configuration Files

### config.el

Main configuration file with custom keybindings, theme settings, and user preferences.

#### Custom Keybindings

```elisp config.el
;; Global keys
(global-set-key [f8] 'neotree-toggle)
(global-set-key (kbd "C-<tab>") 'other-window)
(global-set-key (kbd "M-<down>") 'enlarge-window)
(global-set-key (kbd "M-<up>") 'shrink-window)
(global-set-key (kbd "M-<right>") 'enlarge-window-horizontally)
(global-set-key (kbd "M-<left>") 'shrink-window-horizontally)
```

<AccordionGroup>
  <Accordion title="Keybinding Reference">
    | Key | Action |
    |-----|--------|
    | `F8` | Toggle Neotree file browser |
    | `Ctrl+Tab` | Switch between windows |
    | `Meta+Down` | Enlarge window vertically |
    | `Meta+Up` | Shrink window vertically |
    | `Meta+Right` | Enlarge window horizontally |
    | `Meta+Left` | Shrink window horizontally |
  </Accordion>
</AccordionGroup>

#### User Settings

```elisp config.el
(setq user-full-name "João Pedro"
      user-mail-address "joaopedroaats@pm.me"
      doom-theme 'doom-palenight
      org-directory "~/org/"
      display-line-numbers-type `relative)
```

#### Font Configuration

```elisp config.el
(setq doom-font (font-spec :family "monospace" :size 16)
     doom-variable-pitch-font (font-spec :family "monospace" :size 16)
     doom-big-font (font-spec :family "monospace" :size 24))
```

#### Plugins

```elisp config.el
;; Neo theme icons
(setq neo-theme (if (display-graphic-p) 'icons 'arrow))

;; Waka time tracking
(global-wakatime-mode)
```

## Customization

### Changing the Theme

Edit `config.el` and modify the `doom-theme` variable:

```elisp
(setq doom-theme 'doom-palenight)  ; Current theme
;; Other options:
;; 'doom-one, 'doom-vibrant, 'doom-molokai, 'rebecca
```

### Adding More Languages

Edit `init.el` in the `:lang` section and add desired language modules:

```elisp
:lang
python            ; Enable Python support
rust              ; Enable Rust support
go                ; Enable Go support
```

Then run `doom sync` to apply changes.

### Installing Additional Packages

Add to `packages.el`:

```elisp
(package! your-package-name)
```

Run `doom sync` and restart Emacs.
